# 🌱 AGMS - Automated Greenhouse Management System

## 📌 Description
Microservices-based system to monitor and automate greenhouse conditions.

## ⚙️ Technologies
- Spring Boot
- Spring Cloud (Eureka, Gateway, Config)
- MySQL

## 🚀 Services
- Config Server
- Eureka Server
- API Gateway
- Zone Service
- Sensor Service
- Automation Service
- Crop Service

## ▶️ How to Run
1. Start Config Server
2. Start Eureka Server
3. Start API Gateway
4. Start all Microservices
